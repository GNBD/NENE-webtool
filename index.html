About NENE Tool

서비스 소개

NENE Tool은 사용자의 편의와 데이터 보안을 위해 제작된 웹 유틸리티입니다. 복잡한 설치 없이 브라우저 내에서 즉시 사용 가능한 도구들을 제공합니다.

보안 약속 (NO Server)

귀하가 입력하거나 편집하는 모든 데이터(이미지, 텍스트 등)는 어떠한 경우에도 외부 서버로 전송되거나 저장되지 않습니다. 모든 프로세스는 사용자의 웹 브라우저 메모리 내에서만 실시간으로 처리됩니다.

사용된 라이브러리 및 기술

Tailwind CSS

현대적인 UI 디자인과 반응형 레이아웃 구현을 위해 사용되었습니다.

HTML5 Canvas API

Privacy Blur 도구의 이미지 픽셀화(Mosaic) 및 렌더링 핵심 엔진입니다.

Google Fonts (Inter)

높은 가독성을 제공하는 'Inter' 타이포그래피가 적용되었습니다.

Vanilla JavaScript

외부 프레임워크 없이 순수 자바스크립트로 구동되어 가볍고 빠릅니다.

Clipboard API

이미지 및 데이터를 클립보드에 즉시 복사하는 기능을 제공합니다.

File & Blob API

브라우저 로컬 환경에서의 이미지 로드 및 다운로드 처리에 사용됩니다.

Heroicons

직관적인 사용자 경험을 위한 인라인 SVG 아이콘들이 사용되었습니다.

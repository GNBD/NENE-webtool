🙈 NENE Tool - Privacy Focused Web Utility

NENE Tool은 사용자의 소중한 데이터와 프라이버시를 보호하기 위해 제작된 독립형 웹 유틸리티입니다. 모든 작업은 서버 전송 없이 브라우저 내에서 안전하게 처리됩니다.

🔒 핵심 가치 (No Server, 100% Client-Side)

제로 업로드: 편집하는 데이터는 절대로 외부 서버로 전송되지 않습니다.

로컬 프로세싱: 모든 연산은 사용자의 기기 메모리 내에서 실시간 처리됩니다.

데이터 소멸: 페이지를 새로고침하거나 닫는 즉시 작업 데이터는 삭제됩니다.

✨ 주요 기능

1. Privacy Blur (이미지 마스킹)

마우스 드래그로 스크린샷 내 민감 정보 실시간 모자이크 처리

모자이크 크기 조절 및 편집 결과물 클립보드 복사/저장 지원

2. NEIS Byte (글자 수 및 바이트 계산)

나이스(NEIS) 입력 기준에 최적화된 텍스트 분석

공백 포함/제외 글자 수 및 UTF-8 기준 바이트 수 실시간 계산

🛠 기술 스택

UI/Design: Tailwind CSS, Inter Font, Heroicons

Core Engine: Vanilla JavaScript, HTML5 Canvas API

Architecture: Serverless Pure Client-side Web App
